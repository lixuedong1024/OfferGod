# OfferGod 图标说明

由于项目刚创建，暂时没有图标文件。

## 临时解决方案

你可以：

1. 从 boss-helper-main/public/icons/ 复制图标到这里
2. 或者使用任意 PNG 图标（建议尺寸：16x16, 32x32, 48x48, 128x128）

## 图标文件位置

```
public/
  icons/
    16.png
    32.png
    48.png
    128.png
```

## 快速复制命令

```bash
cp ../boss-helper-main/public/icons/* public/icons/
```
