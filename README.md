AI加持的智能求职神器，助你斩获心仪Offer！

## ✨ 主要特性

### 🎨 现代化界面
- 深色主题设计，专业高效
- 清晰的视觉层次
- 流畅的交互体验

### 🤖 多 AI 模型加持
- ✅ **Claude (Anthropic)** - 支持 Claude 3.5 Sonnet、Claude 3 Opus 等
- ✅ **OpenAI 兼容接口** - GPT-4o、GPT-4、DeepSeek 等
- 🔧 自定义 API 端点

### 🚀 核心功能
- 批量投递简历
- AI 智能筛选岗位
- 自动打招呼
- 工作流可视化编排
- 实时活动监控
- 风控状态管理
- 数据统计分析

## 📦 安装

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建
pnpm build
```

## 🔧 配置 AI 模型

### Claude 配置
1. 访问 [Anthropic Console](https://console.anthropic.com/)
2. 创建 API Key
3. 在设置页面添加模型，选择 "Claude (Anthropic)"
4. 填入 API Key 和选择模型版本

### OpenAI 兼容配置
1. 获取 API Key（OpenAI、DeepSeek、Kimi 等）
2. 在设置页面添加模型，选择 "OpenAI 兼容"
3. 填入完整的 API 地址和 Key

## 🎯 使用说明

1. 安装浏览器扩展
2. 访问 Boss 直聘网站
3. 点击扩展图标打开 OfferGod
4. 在设置中配置 AI 模型
5. 设置筛选条件和工作流
6. 开启神之加持，自动化投递

## 🛠️ 技术栈

- **框架**: Vue 3 + TypeScript
- **构建**: WXT (Web Extension Tools)
- **UI**: Element Plus + 自定义 CSS
- **状态管理**: Pinia
- **AI SDK**: 
  - @anthropic-ai/sdk (Claude)
  - openai (OpenAI 兼容)

## 📝 特色功能

### 神之加持模式
- **保守模式**: 每日 ≤ 40 次，稳扎稳打
- **平衡模式**: 每日 ≤ 80 次，效率与安全兼顾
- **激进模式**: 每日 ≤ 150 次，全力冲刺

### AI 智能筛选
- 自动分析岗位匹配度
- 智能过滤不合适岗位
- 生成个性化打招呼语

### 实时监控
- 投递进度实时显示
- 风控状态智能预警
- 数据统计一目了然

## ⚠️ 注意事项

- 本项目仅供学习交流使用
- 使用自动化工具存在风险（账号权重降低、封号等）
- 请合理控制投递频率
