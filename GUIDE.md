# Jobflow Enhanced 使用指南

## 快速开始

### 1. 安装扩展

```bash
# 克隆项目
cd jobflow-enhanced

# 安装依赖
pnpm install

# 开发模式（Chrome）
pnpm dev

# 构建生产版本
pnpm build
```

### 2. 加载到浏览器

**Chrome/Edge:**
1. 打开 `chrome://extensions/`
2. 开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择 `.output/chrome-mv3` 目录

**Firefox:**
1. 打开 `about:debugging#/runtime/this-firefox`
2. 点击"临时载入附加组件"
3. 选择 `.output/firefox-mv3/manifest.json`

## AI 模型配置

### Claude (推荐)

Claude 在理解中文和生成自然对话方面表现优秀，特别适合：
- 岗位匹配度分析
- 生成个性化打招呼语
- 智能筛选简历

**配置步骤：**

1. 访问 [Anthropic Console](https://console.anthropic.com/)
2. 注册/登录账号
3. 创建 API Key
4. 在 Jobflow 设置页面点击"添加模型"
5. 选择"Claude (Anthropic)"
6. 填写配置：
   - **模型名称**: 自定义名称，如"我的 Claude"
   - **API Key**: 以 `sk-ant-` 开头的密钥
   - **模型**: 推荐 `Claude 3.5 Sonnet (最新)`
   - **Temperature**: 0.7（默认，可调整创造性）
   - **Max Tokens**: 4096（默认）

**模型选择建议：**
- **Claude 3.5 Sonnet**: 最佳性价比，速度快，质量高
- **Claude 3 Opus**: 最强能力，适合复杂分析
- **Claude 3 Haiku**: 最快速度，适合简单任务

### OpenAI 兼容接口

支持所有 OpenAI 兼容的 API，包括：

**官方 OpenAI:**
- URL: `https://api.openai.com/v1/chat/completions`
- 模型: `gpt-4o`, `gpt-4o-mini`, `gpt-4`

**DeepSeek (推荐国内用户):**
- URL: `https://api.deepseek.com/chat/completions`
- 模型: `deepseek-chat`
- 优势: 价格便宜，中文友好

**Kimi (月之暗面):**
- URL: `https://api.moonshot.cn/v1/chat/completions`
- 模型: `moonshot-v1-8k`, `moonshot-v1-32k`

**智谱 AI:**
- URL: `https://api.zhipuai.cn/v1/chat/completions`
- 模型: `glm-4`, `glm-4-plus`

**配置步骤：**
1. 在设置页面点击"添加模型"
2. 选择"OpenAI 兼容"
3. 填写完整的 API URL
4. 填写 API Key
5. 选择或输入模型名称

## 功能使用

### 工作台 (Dashboard)

实时监控自动化运行状态：
- **今日投递**: 当天已投递数量
- **累计投递**: 总投递数
- **回复率**: HR 回复比例
- **面试率**: 获得面试机会比例
- **实时活动**: 查看每一步操作日志
- **风控状态**: 监控账号安全

### 岗位列表 (Jobs)

管理和筛选岗位：
- **全部**: 查看所有岗位
- **待投递**: AI 筛选通过，等待投递
- **已投递**: 已发送简历
- **已回复**: HR 已回复

每个岗位显示：
- 匹配度评分（AI 计算）
- 公司名称、岗位、薪资
- 工作地点、经验要求
- 当前状态

### 工作流 (Workflow)

可视化配置自动化流程：

**默认工作流：**
```
触发器 → AI 筛选 → 投递简历
              ↓
            跳过
```

**节点类型：**
- **触发器**: 定时或手动触发
- **AI 筛选**: 使用 AI 分析岗位匹配度
- **投递简历**: 自动发送简历
- **打招呼**: 发送个性化消息
- **条件判断**: 根据规则分流

### 设置 (Settings)

**AI 模型管理：**
- 添加/删除模型
- 查看模型配置
- 切换默认模型

**筛选条件：**
- 薪资范围
- 工作地点
- 经验要求
- 公司规模
- 行业类型

**自动化参数：**
- 投递频率（保守/平衡/激进）
- 每日上限
- 操作间隔
- 风控策略

## 使用技巧

### 1. 合理设置投递频率

**保守模式（推荐新手）：**
- 每日 ≤ 40 次
- 间隔 5-10 分钟
- 匹配度 ≥ 85

**平衡模式（推荐）：**
- 每日 ≤ 80 次
- 间隔 2-5 分钟
- 匹配度 ≥ 75

**激进模式（谨慎使用）：**
- 每日 ≤ 150 次
- 间隔 30-90 秒
- 匹配度 ≥ 65

### 2. 优化 AI 提示词

在工作流的 AI 筛选节点中，可以自定义提示词：

```
你是一个专业的求职顾问。请分析以下岗位是否适合候选人：

候选人背景：
- 3年前端开发经验
- 熟悉 React、Vue、TypeScript
- 期望薪资：25-35K
- 期望地点：北京、上海

岗位信息：
{{job_title}}
{{company_name}}
{{salary}}
{{requirements}}

请从以下维度评分（0-100）：
1. 技能匹配度
2. 薪资匹配度
3. 发展前景
4. 公司规模

最后给出综合评分和是否推荐投递。
```

### 3. 监控风控状态

注意以下指标：
- **今日操作数**: 不要超过设定上限
- **平均间隔**: 保持在合理范围
- **异常次数**: 如果出现异常，立即暂停

### 4. 个性化打招呼

使用 AI 生成个性化消息：

```
你好，我是一名有{{experience}}年经验的{{position}}。
看到贵公司的{{job_title}}岗位，我对{{highlight}}特别感兴趣。
我在{{skill}}方面有丰富经验，期待能有机会详聊。
```

## 常见问题

### Q: Claude API 调用失败？
A: 检查：
1. API Key 是否正确
2. 是否有足够的额度
3. 网络是否能访问 api.anthropic.com
4. 可以尝试使用代理或中转服务

### Q: 投递没有生效？
A: 可能原因：
1. 页面没有滚动到底部
2. 达到每日上限
3. 被 Boss 直聘限流
4. 检查浏览器控制台错误

### Q: AI 筛选不准确？
A: 优化方法：
1. 调整 Temperature 参数（降低随机性）
2. 优化提示词，提供更多上下文
3. 尝试不同的模型
4. 增加筛选条件的权重

### Q: 如何避免封号？
A: 建议：
1. 使用保守或平衡模式
2. 不要在短时间内大量操作
3. 定期暂停，模拟人工行为
4. 避免在非工作时间投递
